# Website Usage Control Extension

Internet Programming Sessional Project

## Team Goal
Build a Chrome extension that tracks website usage, shows analytics, and allows blocking distracting sites.

## Features (Planned)
- Website usage tracking
- Daily usage analytics
- Focus mode
- Website blocking
- Dashboard

## Project Structure
background.js -> extension core logic  
modules/ -> tracking, blocking, storage  
ui/ -> popup and dashboard  

## Branch Strategy
main -> stable version  
dev -> integration branch  
feature/* -> development branches
